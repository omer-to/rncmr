**_Core Component: View_**

    *Default View styles:

```javascript
{
    flex: 0,
    flexDirection: 'column',
    justifyContent: 'flex-start',
    alignItems: 'stretch'
}
```

    A *View* component can have zero to many children.





    How to draw a circle shape using a *View* component:

```javascript
{
    width: size,
    height: size,
    borderRadius: size / 2
}
```
